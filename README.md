# System Brain — landing page

Public marketing site for the managed AI service: one memory layer, connected systems, custom dashboards, and automation.

**Live site:** https://undisputednoobs.com/system-brain/ (also https://stevew00dy.github.io/system-brain/)

## Local preview

```bash
python3 -m http.server 8765
```

Open http://127.0.0.1:8765/

## Before publishing updates

1. Replace `hello@systembrain.co` in `index.html` with your email or Calendly link.
2. Edit brand copy in `index.html` if needed.

## Stack

Static HTML + CSS + minimal JS. No build step. Deployed via GitHub Pages from `main`.
